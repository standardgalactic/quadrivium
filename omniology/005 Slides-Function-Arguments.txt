Function Arguments
1. Positional Arguments
 
 
●
The most common way of assigning arguments to parameters is via the order in which they are 
passed, or their position.
2. Default Arguments
 
 
●
A positional argument can be made optional by specifying a default value for the corresponding 
parameter when defining the function.
3. Keyword Arguments 
●
Keyword arguments allow us to ignore the order in which we entered the parameters in the 
function definition or even to skip some of them when calling the function.
4. *args 
●
Used when you don't know how many parameters you're going to need in a function(variable 
number of parameters)  
5. **kwargs 
 
●
Used when you have a variable number of keyword arguments
●
Kwargs stands for keyword arguments and it builts a dictionary of key: value pairs.
