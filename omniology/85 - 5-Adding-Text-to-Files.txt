Adding Text to Files: 
 
 
echo command 
 
echo “Your text goes here” > filename 
(To add text and create a new file) 
 
echo “Additional text” >> filename 
(To append to an existing file) 
 
cp command 
 
cp exisiting-file new-filename 
(To copy an existing file to new file) 
 
cat existing-file > new-filename  (cat the content of an existing file and add to new file.  This 
command does the same as above) 
 
vi command 
 
vi filename 
(Create a new file and enter text using vi insert mode)
