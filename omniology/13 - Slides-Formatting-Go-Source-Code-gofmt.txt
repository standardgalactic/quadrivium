Formatting Go Source Code (gofmt)
●
Go strongly suggests certain styles.
●
gofmt which comes from golang formatter will format a program's source code in an 
idiomatic way that is easy to read and understand. 
●
This tool, gofmt,  is built into the language runtime, and it formats Go code according to a 
set of stable, well-understood language rules. 
●
We can run it manually at the command line or we can conﬁgure our IDE, for example 
VSCode, to run gofmt each time a ﬁle is saved.
Example: gofmt -l -w main.go
