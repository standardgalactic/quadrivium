\[
\int u^{-1/4} \, du
\]

Solving this integral involves increasing the exponent by 1 and dividing by the new exponent:

\[
\int u^{-1/4} \, du = \frac{u^{3/4}}{3/4} + C = \frac{4}{3} u^{3/4} + C
\]

Substituting back \(u = x^2 + 1\), the solution becomes:

\[
\frac{4}{3} (x^2 + 1)^{3/4} + C
\]

Therefore, the indefinite integral of \((2x) / (x^2 + 1)^{-1/4}\) with respect to \(x\) is:

\[
\frac{4}{3} (x^2 + 1)^{3/4} + C
\]
