Here's a concise summary of the instructions for installing and using STAT Plus, a free statistical analysis software compatible with macOS:

1. **Accessing STAT Plus:**
   - Use Google Chrome to navigate to "STAT plus" on analystsoft.com to access the download page.

2. **Downloading and Installing:**
   - Download the Mac space LE version from the site.
   - Find the downloaded file, open it by double-clicking, and follow installation steps, typically involving moving the app to the Applications folder.

3. **Using STAT Plus with Excel:**
   - Open STAT Plus from the Applications folder, which also launches Microsoft Excel in the background.
   - Work on new or existing spreadsheets for statistical analysis, switching between applications as needed using mouse buttons.

4. **Statistical Analysis:**
   - Perform a range of analyses similar to the Windows version; some features require the pro version.
   - Access descriptive statistics and comparisons of means without additional purchases.

5. **Final Steps:**
   - Results will be displayed in your Excel spreadsheet after analysis.
   - The setup supports following along with a lecture series that uses this software on Mac-compatible Microsoft Excel.
